Change Log - v1.0

Since last GitHub Pages upload

  • Added editable generated-note panel with manual edit highlighting and copy flow improvements.
  • Added undo support for builder actions and improved reset/default save workflow.
  • Added homepage settings cog and global reset action for all saved template defaults.
  • Refined landing experience with pathway navigation for Exam, Extraction, Crown, and Periodontal templates.
  • Standardized wording system across templates to reduce wording drift and improve consistency.
  • Removed item code rows from templates and defaults normalization.
  • Upgraded tooth picker to support multi-tooth plus per-tooth surface selection with canonical surface ordering.
  • Updated caries and leaking/defective restoration pickers to use tooth plus surfaces where required.
  • Linked diagnosis output to selected caries/restorative findings with correct Nil fallbacks.
  • Expanded OHI and This Visit controls in Exam and Clean with cleaner, selectable output lines.
  • Added composite material quick options for CR restorations.
  • Standardized LA blocks across templates, including IDB side selection and site/agent/volume details.
  • Added optional Prescriptions sections for Endo, Extraction, Periodontal, and Freestyle templates.
